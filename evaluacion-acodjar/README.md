# Sistema de Evaluación ACODJAR - Guía de Usuario

Esta es una guía interactiva web para el sistema de evaluación del Departamento de Comunicaciones de ACODJAR de R.L.

## Características

- ✅ Explicación de los 10 criterios de evaluación
- ✅ Sistema de ponderación (60% evaluación de compañeros, 40% autoevaluación)
- ✅ Guía paso a paso para usuarios
- ✅ Gráficos interactivos
- ✅ Sección de preguntas frecuentes
- ✅ Solución de problemas comunes

## Cómo desplegar en Vercel

1. Crea una cuenta en [Vercel](https://vercel.com)
2. Instala Vercel CLI: `npm install -g vercel`
3. En la carpeta del proyecto, ejecuta: `vercel`
4. Sigue las instrucciones en pantalla

## Desarrollo Local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Tecnologías Usadas

- Next.js 16
- React 19
- Tailwind CSS
- Radix UI
- Recharts (para gráficos)
- TypeScript
